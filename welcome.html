Welcome to the ConvertMsgToEml installer


This installer can automatically setup a dummy Mail account to assist with viewing of the resulting converted files. In order to enable this, you must give Full Disk Access to the Installer.

1. Open System Settings
2. Select Privacy & Security
3. Expand Full Disk Access
4. Enable Installer

If Installer is not listed, click the "+" at the bottom of the frame to add it. In the Finder window, search for Installer.app, select the iOS App Install and then click Open.
